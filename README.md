# The Greatest Repository in the World

> This will be the greatest project in the world — but we lack a direction, a plan, and any purpose.
> **You are the chosen one. Your PR is the direction.**


## What Is This

Zero code. Zero vision. Zero clue.

This is the most ambitious project on GitHub. We have no roadmap, no tech lead — only your PR. It is an open, collaborative social experiment.

No one knows what it will become. It all depends on your PR.

## The Only Rule

**All non-malicious Pull Requests will be merged.**

As long as your PR doesn't try to hurt anyone or sabotage the experiment, it gets merged. You are the engineer of this grand blueprint.

## What Counts as a Malicious Pull Request?

Even the greatest project needs a tiny bit of order. The following will be considered malicious:

- **Modifying this README**
- Any code that harms computers
- Hate speech, discrimination, harassment
- Explicit content
- Spam or ads
- Copyright-infringing files
- Pointless mass deletion of others' contributions

## How to Contribute to the Greatest Repository in the World

1. Fork it.
2. Add or change anything.
3. **Open a glorious Pull Request.**
4. Wait for my Merge.

*No coding skills required. As long as you can type a single character.*

## A Note on Safety

- GitHub Actions have been disabled. No code here will run automatically.
- Do not download and run something a stranger on the internet threw in here.
- If you spot something malicious, open an Issue.

---

*Thank you for contributing to the greatest repository in the world.*

# 世界上最伟大的仓库

> 这将是世界上最伟大的项目——但我们缺少一个方向、一个计划和任何目的。
> **你就是天选之子。你的 PR 就是方向。**

## 这是什么

零代码、零愿景、零头绪。

这是 GitHub 上最有野心的项目。我们没有思路，没有技术主管，只有你的 PR 。这是一场开放的协作式社会实验。

没有人——知道它最终会变成什么。全看你的 PR。

## 唯一的规则

**所有非恶意的 Pull Request 都会被合并。**

只要你的 PR 不想伤害任何人，也不会毁掉这个实验，它就能被合并。你就是伟大蓝图的工程师。

## 什么叫做恶意的 Pull Request ？

就算是最伟大的项目，也需要一丁点规则。以下内容会被视为恶意：

- **修改本 README 文件**
- 任何会伤害电脑的代码
- 仇恨言论、歧视、骚扰内容
- 露骨的内容
- 垃圾广告
- 侵权文件
- 无意义地大量删除他人贡献

## 如何做贡献

1. Fork 它。
2. 添加或修改任何东西。
3. **提交一个伟大的 Pull Request** 。
4. 等待我的 Merge 。
*无需任何编程技能。只要你会敲一个字符。*

## 安全须知

- GitHub Actions 已被我禁用。这里里没有任何代码会自动执行。
- 不要下载并运行一个网上陌生人塞进来的东西。
- 如果你发现了恶意内容，请开个 Issue。
